1. LE TEST DE LA CAUSTIQUE

Le Problème de l'Aberration Sphérique

Un miroir sphérique focalise tous les rayons en un point unique (le centre de courbure). Mais pour un télescope, nous utilisons souvent des formes Paraboliques ou Hyperboliques pour corriger les images venues de l'infini.

Ces miroirs asphériques n'ont pas un rayon de courbure constant. Le "centre" varie selon la zone du miroir. L'ensemble de ces centres forme une courbe appelée La Caustique.

La Méthode Platzeck-Gaviola

Ce logiciel utilise les équations dérivées par Cornejo & Malacara (1978) pour calculer où placer l'appareil de contrôle (Foucault) pour chaque zone du miroir.

Données d'Entrée : R (Rayon), K (Constante), S (Zone)

2. LES FORMULES MATHÉMATIQUES

Voici comment le logiciel fonctionne "sous le capot". Ces équations sont les solutions analytiques exactes (Corrigées v83).

Étape 1 : La Courbure (c)

c = 1 / R

Étape 2 : La Flèche (z) - "Sagitta"

C'est la profondeur du miroir à la hauteur S.

z = (c * S²) / (1 + √(1 - (K+1) * c² * S²))

Étape 3 : Coordonnées de la Caustique (X, Y)

Le logiciel calcule deux valeurs. Y est la position longitudinale (avant/arrière) et X est la position transversale (latérale).

Terme A = c * z * (K + 1)

Y = -K * z * [3 + A * (A - 3)]
X = -S * c * K * z * [2 + A * (A - 3)] / (1 - A)

3. EXEMPLE DE CALCUL À LA MAIN

Prenons un miroir standard : R = 6200 mm, K = -1, S = 250 mm

1. Calcul de c : c = 1 / 6200 ≈ 0.00016129

2. Calcul de z : Pour K = -1, formule simplifiée : z = c * S² / 2 = 250² / (2 * 6200) ≈ 5.0403 mm

3. Calcul de A : Comme K+1 = 0, alors A = 0.

4. Calcul de Y : Y = -(-1) * 5.0403 * [3 + 0] = 15.1209 mm

5. Calcul de X : X = 500 * (1/6200) * 5.0403 * 2 / 2 (correction) ≈ 0.4064 mm

4. MODULE ARCHITECTE OPTIQUE

Ce module utilise la théorie de Schwarzschild du 3ème ordre pour calculer les miroirs.

Dall-Kirkham

Pour un DK, on impose un secondaire sphérique (K2=0). Le primaire doit être une Ellipse Prolate pour compenser.

K1 = -1 + [ (M² - 1) / M³ ] * (Distance Secondaire-Foyer / Focale Primaire)
